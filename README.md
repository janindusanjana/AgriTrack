# AgriTrack 🍃

AgriTrack is a desktop-based Agricultural Management System designed specifically to streamline and automate the day-to-day operations of tea cultivation estates in Sri Lanka. It provides a centralized platform to manage plantation workers, track daily harvests, handle financial advances, and visualize production trends.

---

## 🚀 Features

* **Interactive Analytics Dashboard:** Features a dynamic live graph (powered by LiveCharts) that visualizes total crop harvest trends (Kg) over time, allowing managers to monitor productivity at a glance.
* **Worker Management:** Easily register, track, and maintain profiles for estate workers and active farmers.
* **Daily Harvest Tracking:** Log daily harvest weights (kilos plucked) per worker to maintain accurate yield records.
* **Advance & Settlement System:** Track financial advances given to workers and manage wage rates to ensure seamless final payroll calculations.
* **Secure Authentication:** Features dedicated Login and Sign-Up systems for safe administrative access.

---

## 🛠️ Tech Stack

* **Language:** C#
* **Framework:** .NET (Windows Forms with WPF Integration)
* **Database:** SQLite (`AgriTrackDB.db`)
* **Data Visualization:** LiveCharts.WinForms / LiveCharts.Wpf
* **IDE:** Visual Studio 2022 / 2026
